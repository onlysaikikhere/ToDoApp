# Simple To-Do

A minimal, single-screen Android to-do app.

## Philosophy

One screen. One list. One action.

No tabs. No analytics. No gamification.

## Features

- Add tasks
- Mark tasks as done
- Delete tasks
- Persist locally with Room

## Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Room Database**
- **MVVM Architecture**
- Target: Android 8+

## Build

Open in Android Studio and run.

## What This App Does NOT Have

- Login
- Cloud sync
- Reminders
- AI
- Streaks
- Categories
- Priorities
- Due dates

By design.
